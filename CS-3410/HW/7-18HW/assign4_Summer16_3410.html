Description of Program

You are to write a program name palindrome.java that reads input from a file and output all of the words in that file that are palindromes. The program will prompt the user for the input file name from which it must read the input.

The program should read the file word by word then extract from each word each character by character, inserting these characters into a queue. Then copy the content of the queue into a stack. After these two processes, it must now pop character by character from the queue and stack and compare them for equality. If the two characters are equal, then keep comparing until both containers are empty. At the end, return that word with the message that the word is a palindrome.

  • Upon extracting words - Note:
    1. Two words are spearated by one or more space.
    2. The end of line character.
    3. a punctuation such as (.,!?, etc.)
  • Convert all characters to either upper or lower case before comparison
  • Remove all punctuations as the last character of any word.
Finally display on the screen each word from the file that is a palindrome.
